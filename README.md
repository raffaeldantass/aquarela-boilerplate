#### Getting Started

First, git clone:

```
git clone git@github.com:raffaeldantass/my-next-boilerplate.git && cd my-next-boilerplate
```

Then, run

```
npm run install
```

And finally:

```
npm run dev
```

#### Build

Just run

```
npm run build
```

and to test the build

```
npm run start
```
